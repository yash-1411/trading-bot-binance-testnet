# Binance Futures Testnet Trading Bot

A Python trading bot for placing market and limit orders on Binance Futures Testnet (USDT-M) with structured code, comprehensive logging, and enhanced CLI experience.

## Features

✅ **Core Functionality**:
- Place MARKET and LIMIT orders on Binance Futures Testnet
- Support for BUY and SELL orders
- Input validation and error handling
- Structured logging to file and console
- Clean separation of concerns (API layer, CLI layer, validation)

✅ **Bonus Features**:
- Enhanced CLI with interactive mode and prompts
- Comprehensive account information display
- Beautiful formatted output

## Prerequisites

1. **Python 3.8+**
2. **Binance Futures Testnet Account**:
   - Register at: https://testnet.binancefuture.com
   - Generate API credentials: https://testnet.binancefuture.com/futures/myAccount/apiManagement
3. **Testnet USDT**:
   - Get testnet funds from the Binance Futures Testnet faucet
